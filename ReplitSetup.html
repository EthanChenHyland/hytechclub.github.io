Replit Setup

Replit.com is a website that allows developers to write and run code right from a web browser! Create a Replit account in order to save and share code projects.

Note: throughout these instructions, there may be interactive Replit tutorials. Be sure to read through those helpful tips in addition to reading through this guide!

Creating an Account with Email

Follow the steps below to create a Repl account. It is also possible to log into Repl with Google or GitHub - feel free to do that instead.

  2. Click the "Continue with email âž¡" link

  7. If necessary, prove your humanity to verify the account
  8. Walk through the onboarding process, or skip if desired
  9. Check your email inbox for a message - and make sure to verify your email!
